# Shopify AI Storefront Generator (GÜNCELLENDİ: 16:15)

Yapay zeka destekli Shopify tema oluşturucu.

## Kurulum

Bu proje Node.js gerektirir.

1.  Bağımlılıkları yükleyin:
    ```bash
    npm install
    ```
    
2.  Geliştirme sunucusunu başlatın:
    ```bash
    npm run dev
    ```

3.  Tarayıcınızda açın: [http://localhost:3000](http://localhost:3000)

## Özellikler

- **AI Üretimi**: Doğal dil ile Shopify tema kodları oluşturun.
- **Dosya Yapısı**: Shopify standartlarına uygun (`/layout`, `/sections`, `/templates`) klasörleme.
- **Kopyalama**: Tek tıkla kodları kopyalayıp Shopify Edit Code alanına yapıştırın.
- **Geçmiş**: Eski tasarımlarınızı görüntüleyin.

## Teknolojiler

- Next.js 14
- TypeScript
- Supabase (Auth + Database)
- Vanilla CSS
